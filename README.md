
<p>
    Após não ter tido o tempo que queria para acompanhar a Semana Omnistack 11 da RocketSeat (tendo que baixar as aulas do YouTube para terminar o projeto), termino a aplicação neste dia (04/05/2020). 

    E que prazer foi o de construir esse app, que tem como objetivo ser um local onde Organizações Não Governamentais (ONG's) podem cadastrar "pedidos de ajuda" e os usuários podem entrar em contato para ajudar financeiramente.

    Estudamos e implementamos Node.js no ambiente Back-End, React.js no Front-End, e React Native no Mobile. Além disso, aprendemos a utilizar o GitHub em sincronia com o VS Code, e,  por fins de comodidade e simplificação, aprendemos e utilizamos o banco de dados SQLite.

    Muito obrigado á RocketSeat por esse projeto fantástico (pra variar), foi mais um prazer participar dessa fantástica semana.
